# Songs Lyrics Finder (React + Vite)

Small React app to search and display song lyrics, with i18n and previewing top artists + Spotify embeds.

Live demo: https://vercel.com/theodora-tasias-projects/songs-lyricis-finder-react

## What this project does
- Search lyrics by artist + song using the lyrics API service in `src/services/lyricsApi`.
- Browse curated `Top Artists` and view each artist's top songs (includes Spotify embed links).
- Internationalization (English + German) using `react-i18next`.
- Responsive styles using SCSS variables and mixins.

## How to run
1. Install dependencies:

```powershell
npm install
```

2. Start dev server:

```powershell
npm run dev
```

3. Build for production:

```powershell
npm run build
```
